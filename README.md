# No-Code-ML-Models-generator
## 1. Main Features: 
1. Upload CSV
2. Use Openai Agent to read csv, ask which x is best and which is y
3. Use agent to ask if x need: onehot encoding, regularize
4. Call train function: split train test, train x_train, create model, save model to db.
5. Download model: query model from db and download in form like pickle
6. Make test prediction: user trys out prediction with created model, input with same row as csv

## 2. Other features:
1. Multilingual system: use i18n to translate apps in multiple language, 
  or use online translator (google, deepl) or AI (chatgpt) to translate 
2. Authentication/Authorization
3. Payment: for pro features
4. Pro features:
    - Unlimitted models creation
    - Visualization
    - Multiple csv files
